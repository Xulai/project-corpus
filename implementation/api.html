API

Our API provides an interface to our datapoints stored in InfluxDB. We also access relational Areas and Devices in MySQL.

We provide endpoints to get the readings, errors and still heres grouped by different time intervals.

Devices and Areas also have each of the datapoints as subresource routes to better represent the resources in the url.

The API also provides Stats overall as well as for devices and area. The stats contain aggregates and counts of the readings.
