API Frontend

Our graphs acted as a proof of concept at the Project Fair as we demonstrated our API's functionality. Using JavaScript framework Vue.js, we created a component for every time period we might be interested in looking at.

We chose to use Vue.js due to its popularity and efficientness when building user interfaces. It's component based structure matched well with the web page we wanted to create and it's documentation was easy to read and learn.
